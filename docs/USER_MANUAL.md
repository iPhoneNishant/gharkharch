## üí≥ Account Management

### Adding Accounts
1. Go to **Accounts** tab
2. Tap **+ Add Account**
3. Choose account type:
   - **Bank Account**: Savings, Current, Credit Card
   - **Cash Account**: Wallet, Petty Cash
4. Enter account details:
   - Account name
   - Opening balance
   - Account number (optional)

### Account Types
- **Bank Account**: Track bank transactions, balance updates
- **Cash Account**: Manage cash flow, petty expenses
- **Credit Card**: Monitor credit card spending and payments

### Account Operations
- View account balance and transaction history
- Edit account details
- Delete empty accounts
- Transfer money between accounts

## üí∞ Transaction Management

### Adding Transactions

#### Manual Entry
1. Go to **Transactions** tab
2. Tap **+ Add Transaction**
3. Select transaction type:
   - **Income**: Salary, Interest, Gift, etc.
   - **Expense**: Food, Transport, Shopping, etc.
4. Choose account
5. Enter amount and description
6. Add category and subcategory
7. Set date and time
8. Save transaction

#### SMS Import (Auto-Detection)
1. Go to **More** ‚Üí **SMS Import**
2. Grant SMS permissions
3. Select bank SMS messages
4. App automatically extracts transaction details
5. Review and confirm imported transactions

### Transaction Categories

**Expense Categories:**

### Transaction Operations
- View all transactions with filters
- Edit transaction details
- Delete transactions
- Search transactions by description
- Filter by date, category, account
- View transaction details

## üîÑ Recurring Transactions

### Setting Up Recurring Transactions
1. Go to **More** ‚Üí **Recurring Transactions**
2. Tap **+ Add Recurring**
3. Enter transaction details
4. Set frequency:
   - Daily
   - Weekly
   - Monthly
   - Quarterly
   - Yearly
5. Set start date and end date (optional)
6. Configure notification preferences
7. Set notification timing (days before due)

### Recurring Transaction Types
- **Fixed Amount**: Same amount each cycle
- **Variable Amount**: Amount changes each cycle
- **Income**: Regular salary, rental income
- **Expense**: Monthly bills, subscriptions

### Managing Recurring Transactions
- View all recurring transactions
- Edit recurring details
- Pause/resume recurring transactions
- Delete recurring transactions
- Receive notifications before due dates
